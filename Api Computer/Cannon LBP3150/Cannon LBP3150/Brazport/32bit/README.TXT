1. Dimensionamento

  Alguns aplicativos podem não imprimir corretamente se o 
  dimensionamento estiver especificado. Nesse caso, defina o 
  dimensionamento como 100%.
  Se o aplicativo tiver sua própria função de dimensionamento, 
  defina-o no aplicativo.

================================================================

2. Layout da página

  Alguns aplicativos dispõem de uma função para imprimir 
  diversos conjuntos de cópias. Entretanto, não use esta função 
  quando uma configuração diferente de [1 página por folha] 
  estiver selecionada em [Layout da página] na guia 
  [Configuração de página].
  Se esta função for usada no aplicativo, os dados poderão não 
  ser formatados corretamente.

================================================================

3. Página separadora

  A [Página separadora] no Windows não pode ser impressa.

================================================================

4. Permissão do usuário para a instalação

  Inicie a sessão como um usuário com direitos de administrador 
  para instalar o driver da impressora. A permissão Controle 
  total é necessária para a instalação.

================================================================

5. Download e instalação de uma impressora de rede 
   (Windows 2000/XP/Server 2003)

  Ao fazer o download e instalar uma impressora de rede no 
  Windows 2000/XP/Server 2003, você pode instalá-la como um 
  usuário sem direitos de administrador.
  Entretanto, se a impressora for instalada por um usuário sem 
  direitos de administrador, algumas funções poderão ficar 
  indisponíveis devido a restrições de segurança do sistema 
  operacional.

  [Funções não disponíveis]
  - O Desinstalador não poderá ser registrado em [Adicionar ou 
    remover programas].

================================================================

6. Instalação em um ambiente multiusuário (Windows XP/Vista)

  Quando esta impressora for instalada em um Windows XP/Vista, 
  não o faça para usuários selecionados pela função "Troca 
  rápida de usuário" do Windows XP/Vista.
  Quando utilizar a função "Troca rápida de usuário", inicie 
  novamente a sessão antes da instalação.

================================================================

7. Desinstalando a impressora de rede que está conectada a outro
   usuário

  Mesmo se você executar o Desinstalador, as informações sobre a
  impressora de rede instalada em um computador local de outro 
  usuário, não são excluídas, mas permanecem em seu sistema 
  operacional como uma informação específica do usuário.
  Se um outro usuário tiver instalado a impressora em seu 
  computador local através da rede, assegure-se de que o usuário
  exclua o ícone da impressora de rede da pasta impressoras 
  antes de você executar o Desinstalador.

================================================================

8. Restrição sobre a impressora de rede

  Quando uma impressora de rede é instalada, mesmo que você 
  altere o destino de conexão para uma porta local, ela não 
  poderá ser usada como impressora local.
  Se você necessitar usar a impressora como uma impressora 
  local, selecione uma porta local e instale novamente o driver 
  da impressora.

================================================================

9. Reiniciando o Windows com um trabalho de impressão pendente

  Caso precise reiniciar o Windows com um trabalho de impressão 
  pendente, faça uma pausa no mesmo na pasta da impressora. Em 
  seguida reinicie o Windows.

================================================================

10. O Modo Protegido do Internet Explorer (Windows Vista)

  Se estiver usando o Windows Vista e executar o Internet 
  Explorer 7 clicando no ícone para [Exibir manual] etc. na 
  Instalação de CD-ROM, o [Modo Protegido] estará definido para 
  [Desativado] independentemente das configurações.
  Se desejar definir o [Modo Protegido] para [Ativado], saia do 
  Internet Explorer 7 e execute-o novamente de forma manual.

================================================================

11. Caracteres para o Nome da impressora (Windows Vista)

  Quando um caractere "@" for utilizado no nome da impressora, 
  essa não será reconhecida pelo sistema operacional por ter um 
  nome de impressora inválido, desabilitando assim a impressão, 
  operação e configuração.

================================================================

12. Registrando Tamanhos de papel personalizado (Windows Vista)

  Verifique se iniciou a sessão como um usuário com direitos de 
  administrador ao editar [Tamanho de papel personalizado] na 
  guia [Configuração de página] no driver da impressora.

================================================================

13. Permissão de acesso às Propriedades da impressora 
    (Windows Vista)

  Ao iniciar a sessãoo como um usuário com direitos de 
  administrador, diferente do que instalou o driver da 
  impressora, não será possível acessar as propriedades da 
  impressora como um usuário com direitos de administrador.
  Para acessar as propriedades da impressora, clique com o botão
  direito do mouse e selecione [Executar como administrador].

================================================================

14. Imprimindo linhas finas (Windows Vista)

  Em alguns aplicativos, a largura da linha pode ser definida 
  para um valor menor do que 1 pt mas a largura real no 
  resultado da impressão pode ser equivalente a uma linha de 1 
  pt.
